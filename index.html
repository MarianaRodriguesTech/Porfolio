Projetos

Seja bem vindo à minha galeria de projetos! Aqui, você encontrará uma amostra do meu trabalho e da minha jornada de aprendizado em desenvolvimento web. Desde os primeiros passos com HTML até a inclusão de interatividade com JavaScript, cada projeto reflete o meu empenho em aprender, criar e aprimorar minhas habilidades. Sinta-se à vontade para explorar e conhecer um pouco mais sobre o que venho desenvolvendo. Espero que esses projetos inspirem e demonstrem meu compromisso com a excelência e o crescimento contínuo.

HTML

  • Formatação de Texto

    Este projeto foi uma introdução à formatação de texto utilizando HTML. Ele apresenta técnicas básicas de formatação, como cabeçalhos, parágrafos e listas.

  • Formulário

    Neste projeto, desenvolvi um formulário simples utilizando HTML. O formulário demonstra o uso de diferentes tipos de entrada de dados, como texto, caixas de seleção e botões de opção.

  • Cafeteria IOS

    Este projeto foi desenvolvido no início da minha jornada com HTML e um pouco de CSS. Ele apresenta uma página de uma cafeteria fictícia chamada Cafeteria IOS, com foco na marcação semântica e na estruturação básica do conteúdo.

CSS

  • Infância

    O projeto "Infância" é uma demonstração do uso de CSS para criar uma galeria de desenhos infantis. Ele utiliza técnicas de layout flexível e estilização para criar uma experiência visualmente atraente.

  • Winx Club

    O projeto "Winx Club" apresenta cards da animação Winx Club utilizando HTML e CSS. Ele demonstra técnicas de posicionamento, estilização e responsividade para criar uma apresentação visualmente atraente dos personagens.

  • Animeta

    Este projeto é uma demonstração de uma página de animação criada com HTML e CSS. Ele apresenta personagens da animação "Divertida Mente" em uma cena interativa, demonstrando habilidades avançadas de design e animação.

JavaScript

  • Escola Dinos

    O projeto "Escola Dinos" é uma aplicação interativa desenvolvida com JavaScript. Ele simula um ambiente escolar demonstrando habilidades avançadas de programação e interatividade.

  • Futuro das linguagens de programação

    Neste projeto, explorei o futuro das linguagens de programação em uma apresentação interativa. Utilizando HTML, CSS e JavaScript, o projeto demonstra habilidades de design e desenvolvimento web.

  • Tic Tac Toe

    O projeto "Tic Tac Toe" é uma implementação do clássico jogo da velha em JavaScript. Ele demonstra habilidades avançadas de lógica de programação e interatividade, oferecendo uma experiência de jogo divertida e desafiadora.
